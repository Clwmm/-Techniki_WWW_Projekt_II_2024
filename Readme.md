### After clone
1. Run ```npm install```

### Start dev Server
1. Run ```npm run devStart```

### Start Tailwind
1. Run ```npx tailwindcss -i ./public/style/input.css -o ./public/style/output.css --watch```

### Database Server:
1. (Atlas)[https://cloud.mongodb.com/]

### Start Server
1. Run ```npm run start```
